Radial Dendrogram

Division >> Battalion >> Station Area

Encoding

The color encodes the number of fire department call for service that went to a given station. The links encode a parent to child relationship from inner to outer (ie. several station areas make up a battalion). The inner nodes do not follow the color scale to prevent misrepresentation of the data.

Interactivity

This visualization support details on demand through tooltips. The tooltip shows the node that is being hovered over. Outer leaf nodes also show the count in the tooltip.

Filtering is also supported with the month input. Select a month in the month input to filter the data being shown in the dendrogram. If no dates are chosen, the dendrogram represents all data from 4/2000 - 4/2020.
